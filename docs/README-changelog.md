# לוג שינויים (CMS Changelog)

קובץ **`docs/cms-changelog.json`** מתעד אוטומטית את כל השינויים שבוצעו במערכת (שמירת מקטע, מחיקה, הוספת תרגום, פרסום ל-Bagel, הוספת/מחיקת נוסח/תרגום/קטגוריה/תפילה/מקטע).

## גישה ללוג

- **במצב פיתוח** (`npm run dev`): כל עדכון ללוג נשמר אוטומטית ב-**`docs/cms-changelog.json`** – אפשר לפתוח את הקובץ ישירות ולצפות בשינויים.
- הקובץ הוא JSON מערך של רשומות; כל רשומה כוללת `timestamp`, `action`, `context`, `details`.

## הערות

- הלוג מתעדכן רק כשהאפליקציה רצה במצב dev (שרת Vite מקומי).
- ייצוא לקובץ טקסט מהקונסולה: `__CMS_CHANGELOG_EXPORT__('text')` או `__CMS_CHANGELOG_EXPORT__('json')`.
